Date Calculator
This C++ program calculates the difference in days between two dates.

Features
Validates the input dates to ensure they are within reasonable bounds according to common calendar rules.
Prints the entered dates along with the corresponding day of the week.
Calculates the difference in days between the two entered dates.
Uses Zeller's Congruence algorithm to determine the day of the week for each date.
Provides a user-friendly interface for inputting dates.
How to Use
Compile the program using a C++ compiler.
Run the compiled executable.
Enter the first date in the format DD.MM.YYYY when prompted.
Enter the second date in the same format when prompted.
The program will display the entered dates along with the corresponding day of the week, and then output the difference in days between the two dates.
